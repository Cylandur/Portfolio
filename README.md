## Featured Projects

### Biblio Britaniana — Cloud-Ready Digital Library Platform

**Tech:** Flask · Python · REST API · PostgreSQL · Docker · Render / Railway / Fly.io

A production-grade digital library management platform built for one-click cloud deployment. Originally developed to manage a personal ebook collection of 1,000+ titles, then generalized into a deployable, cloud-ready application with multi-user support and a fully-featured REST API.

**Key Features:**
- Python library indexer with multi-format support (EPUB, PDF, MOBI, CBZ)
- Full REST API backend with automated cover art fetching and metadata enrichment via Open Library / Google Books
- Multi-format in-browser reader with annotation and reading progress tracking
- Standalone macOS desktop app bundle (no Python install required)
- Configurable demo mode for safe public deployment with sample data
- Complete deployment configuration for Render, Railway, Heroku, and Fly.io
- Docker + docker-compose support for self-hosted deployment

**Key Features:**
